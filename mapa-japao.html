Objetivo

Este projeto tem o intuito de reunir informações sobre as principais regiões do Japão para melhor conhecimento próprio.
É aprendendo sobre novas regiões, suas práticas e costumes é que se aprende mais sobre o país como um todo.

Modo de utilização

Clique em uma região respectiva do mapa para ser redirecionado para a lista de prefeituras pertencentes a ela.
Depois disso, é só clicar na prefeitura que quiser descobrir mais informações.
