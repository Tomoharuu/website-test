Análise de Dados - Lista de Cursos Udemy:

Este projeto tem o objetivo de analisar alguns dados sobre os cursos fornecedos pela plataforma Udemy.
A Udemy é uma plataforma de educação à distância (EAD) que oferece mais de 100 mil cursos on-line gratuitos e pagos com certificação. No site, é possível assistir a aulas em diversas áreas como negócios, artes e gastronomia.

Alguns temas que planejam ser abordados por este projeto são:
- Quais são os cursos que estão em tendência;
- Análise área a área para ver o que está sendo mais requisitado;
- Quão influente é o fato dos cursos serem gratuitos ou pagos;
- Os cursos mais acessados são necessariamente os mais bem avaliados?
- A duração e número de aulas influencia na popularidade do curso?
- Os cursos mais procurados são aqueles com dificuldade mais baixa ou alta?
- Há predominância dos cursos novos ou antigos?

Primeiro passo: Database

Seguirá a baixo um trecho da base de dados que será utilizada para a análise de parâmetros.
Entre os dados coletados da Udemy, estão:
- Nome do Curso;
- Preço do Curso (caso haja);
- Número de Alunos Inscritos;
- Número de Avaliações (Reviews);
- Nível do Curso (Iniciante, Intermediário, Avançado);
- Quantidade de aulas e duração do curso;
- Nota de avaliação dos usuários.


Listagem das áreas atendidas pela Udemy

Os dados coletados referente aos cursos da Udemy vão se referir a 4 tipos de cursos. Sâo eles:
1. Desenvolvimento Web (1203 cursos)
2. Finanças Empresariais (1191 cursos)
3. Instrumentos Musicais (680 cursos)
4. Design Gráfico (602 cursos)


Trabalhando com a base de dados

Dadas as devidas considerações referente aos tipos de dados trabalhados, se inicia aqui a análise propriamente dita referente a essas informações.
Para tal, será levado em conta, a princípio, cada uma das principais áreas da Udemy para identificar quais são os padrões recorrentes a tais cursos, para depois aprofundar o conceito para uma visão mais generalizada.
